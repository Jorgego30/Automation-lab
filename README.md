# Automation Lab 🛠️

A personal repository dedicated to developing, testing, and automating scripts and workflows. This lab focuses on building tools from scratch to understand low-level logic, managing environments cleanly, and practicing script administration.

## 📁 Repository Structure

* **`bots/`**: Scripts for autonomous agents and platform integrations.
    * `telegram-bots/`: Practical implementations and tests using the Telegram API.
      
## ⚙️ Requirements & Core Tools
* **Python 3.x** (For bot development and advanced scripting)
* **Bash / Shell** (For native Linux automation and system management)
